Download the required OpenNeuro datasets and place them in the corresponding `data/` subfolders.

You may need to update the `configs` file to match your local folder names.

---

## Datasets

- **Whole Head Ball-Squeezing (BS)**  
  https://openneuro.org/datasets/ds006903/versions/1.0.0  
  → `data/BS_Laura/`

- **Whole Head Resting-State (RS)**  
  https://openneuro.org/datasets/ds006903/versions/1.0.0  
  → `data/NN22_Resting_State/`

- **Motor Cortex Ball-Squeezing**  
  https://openneuro.org/datasets/ds005930/versions/1.0.1  
  → `data/HD_Squeezing/`
